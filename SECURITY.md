# 安全策略

## 报告安全漏洞

如果发现安全漏洞，**不要**在 GitHub Issues 公开提。请通过以下方式直接联系：

- 在 GitHub 上创建 Security Advisory：
  https://github.com/ASmallMatch/z-wiki/security/advisories/new

我们会尽快确认并修复。

## 受支持的版本

| 版本 | 支持状态 |
|---|---|
| 最新 Release | ✅ 安全更新 |
| 旧版本 | ❌ 不再维护 |

## 安全实践

- 依赖通过 Dependabot 自动更新
- Secret scanning 已启用，防止密钥泄露
- 所有 PR 需经过审查后合并

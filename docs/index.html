Sec. 01 — Features

它替你做什么

01 01 / Ingest

上传自动编译

拖拽 docx、xlsx、pptx、odt、epub、html、csv 进窗口,内置 pandoc 转 markdown,agent 自动编译进知识库,原文归档不丢。

02 02 / Shelf

3D 书架首页

知识库可视化成一座书架,转一转、点一本,直接翻开阅读。明暗两套主题随你换。

03 03 / Read

文章阅读

markdown 渲染、wiki 链接互相跳转,文章之间是连起来的;agent 的思维链折叠展示,不打扰阅读。

04 04 / Agent

对话式追问

直接问知识库:agent 替你读、写、编辑、搜索。可选思考模式,看它怎么一步步找到答案。

05 05 / Vaults

多知识库

工作库、个人库、项目库分开存放,可放自定义位置(如 D 盘),设置里一键切换。

06 06 / Offline

开箱即用

pandoc、ripgrep、fd 已预打进安装包,不联网下载工具。填好 LLM 配置即可开工。

Sec. 02 — How it works

三步,废料变书架

01

拖进去

文档原样归档进 raw/,不预转、不改动。你的原始文件永远是只读真相源。

02

agent 编译

agent 读取原文(pandoc 格式自动转文本),消化、结构化,写进 wiki/,并登记索引与时间线。

03

翻阅与追问

书架上点书翻阅,或直接在对话里追问。新产生的知识再回写知识库,越用越厚。

raw/ 只读归档 · wiki/ 结构化知识 · index.md 索引 · log.md 时间线 · output/ 报告

Sec. 03 — Download

拿一个回去

全平台安装包,从 Releases 页获取 → github.com/ASmallMatch/z-wiki/releases
平台安装包用法
Windows zwiki-setup-*.exe 安装版 / *-win.zip 便携版 安装版双击装;便携版解压后双击 z-wiki.exe
macOS *-arm64.dmg Apple Silicon / *.dmg Intel 双击 dmg,把 z-wiki 拖进「应用程序」
Linux *.AppImage chmod +x 后双击或命令行运行
首次启动较慢属正常 —— 首次要把内置 pandoc(约 180MB)复制到用户数据目录,等几十秒到 1 分钟,不是卡死。第二次起就快了。 启动后到「设置」填 LLM 配置(Base URL / 模型 / API Key),agent 才能调用。
Sec. 04 — Local-first

你的知识,在你手里

数据在本机。知识库内容、配置、对话历史全部落在你的磁盘上,重装、升级不丢。

server 走 loopback。桌面 app 内嵌的服务只监听本机回环地址,不对外暴露;除了你配置的 LLM 接口,什么都不上传。

Knowledge base

默认在应用数据目录下的 kb/,可换到任意位置(如 D 盘),多 Vault 切换。

Config

配置与 agent 资源在系统用户目录(%APPDATA% / ~/Library / ~/.config),与安装目录分离。

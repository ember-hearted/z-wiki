# 01: useTheme 多实例不同步导致主题切换时书架不换皮

- 状态:`ready-for-human`
- 关联 ADR:`docs/adr/0005-theme-system.md`(主题系统)

## 现象

首页明暗主题切换时,3D 书架的书本颜色(封面/书脊/书顶/书边)整体不变。`document.documentElement.dataset.theme` 会变(背景等 CSS 驱动元素随之变化),但 `BookShelf3D` 的 `theme` prop 不更新,换皮副 `useEffect([theme])` 不触发。

## 根因

`useTheme` hook 每次调用创建独立的 `useState`,无 React Context 共享。`ThemeToggle`(在 `Header` 内)与 `Home` 各自调用 `useTheme()`,得到两份独立 state:

- `ThemeToggle` 调 `toggle` 只改自己那份 state,并经 `useEffect` 写 `data-theme` DOM → CSS 变量变 → 背景变。
- `Home` 那份 state 不变 → `<BookShelf3D theme={theme} />` prop 不变 → 副 `useEffect([theme])` 不触发 → 书不换皮。

`BookShelf3D` 的换皮逻辑(ADR-0006 D2')本身正确,问题纯在 `useTheme` 无全局共享。

## Agent Brief

> *This was generated by AI during triage.*

**Category:** bug
**Summary:** `useTheme` 改为 React Context 共享,使 `ThemeToggle` 与 `Home`(及所有消费者)读到同一份 theme state。

**Current behavior:**
`useTheme()` 每次调用产生独立 `useState`。`ThemeToggle` 与 `Home` 各调一次,两份 state 不同步。切换时 `ThemeToggle` 的 state + DOM 更新,但 `Home` 的 state 不变,`BookShelf3D` 的 `theme` prop 不更新,换皮副 `useEffect([theme])` 不触发。

**Desired behavior:**
所有 `useTheme()` 调用共享同一份 theme state。`ThemeToggle` 切换后,`Home` 及任何消费者收到的 `theme` 同步更新,`BookShelf3D` 副 `useEffect([theme])` 触发换皮。初始值仍读 `document.documentElement.data-theme`(index.html FOUC 内联脚本已设),保证首屏不二次跳动;DOM 同步与 localStorage 持久化逻辑保留。

**Key interfaces:**
- `useTheme` hook 返回类型 `{ theme: Theme; toggle: () => void }` 不变,但实现改为 `useContext`。
- 新增 `ThemeProvider` 组件,持有唯一 `useState<Theme>`,提供 Context value,内含现有 `useEffect`(写 `data-theme` + localStorage)与 `toggle`。
- `useTheme` 在无 Provider 时抛错(防误用)。
- `App` 顶层用 `ThemeProvider` 包裹所有子树(`Header` + `main` + `FloatingActions` + `ChatDrawer`),确保 `ThemeToggle` 与 `Home` 在同一 Provider 下。

**Acceptance criteria:**
- [ ] `useTheme` 基于 Context,所有消费者共享同一 state。
- [ ] `App` 顶层包 `ThemeProvider`,覆盖 `ThemeToggle` 与 `Home`。
- [ ] 切主题时 `BookShelf3D` 的 `theme` prop 更换,书皮换皮生效。
- [ ] 首屏初始主题仍由 index.html FOUC 脚本决定,无二次跳动。
- [ ] localStorage 持久化与隐私模式静默降级行为保留。
- [ ] `make typecheck` 与 `make format` 通过。
- [ ] 无回归:非首页(如 Article/Settings)的主题切换与样式不变。

**Out of scope:**
- 改动 `BookShelf3D` 换皮逻辑(它本身正确)。
- 改动 CSS 变量或 `--surface-shelf` 等。
- 新增主题或调整 archive/draft 配色。
